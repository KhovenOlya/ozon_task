## **Задача**
Разработать прототип блока Progress для использования в мобильных web-приложениях. Основное предназначение блока отображать процесс выполнения процессов и их прогресс выполнения.

## Описание
Класс Progress реализует интерактивный круговой прогресс с управляемыми параметрами. Он объединяет три основные функции: изменение значения, включение анимации и скрытие круга.

### Методы класса
1. setValue()
- Устанавливает значение прогресса в диапазоне от 0 до 100.  
- Обновляет внутреннее состояние и визуальное отображение круга.  
2. setAnimate()
- Управляет включением или отключением анимации.  
- Вызывает функцию `animateCircle`, которая запускает или останавливает вращение круга. 
3. setHide()
- Управляет скрытием круга.  
- Вызывает функцию `hideCircle`, которая удаляет или восстанавливает круг.  

### Реализация
- Для управления визуализацией и анимацией использованы отдельные модули: `value.js`, `animate.js`, `hide.js`.  
- Блок реализован с адаптивной версткой, корректно отображается как на мобильных устройствах, так и на десктопах.
